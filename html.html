html이란 무엇인가

html이란 웹페이지를 제작하는데 사용되는 언어이다. 웹 상에 구현되는 것은 모두 html로 이루어져 있다.
프로그래밍 언어 중 가장 쉽지만, 가장 중요한 언어이다.

실습환경 조성하기

필요한 것은 웹브라우저와 에디터이다. 웹브라우저는 되도록 크롬을 사용하자.
에디터는 아톰(atom)을 사용할 것이다. 아톰 공식홈페이지에서 다운받을 수 있다.
프로젝트용 폴더를 하나 새로 만들고 File-Open Folder...를 하면 기본 세팅이 끝난다.

기본문법, '태그'

html은 기본 문법인 태그로 이루어져 있다. 태그의 종류는 두가지다.
태그는 1.내용을 감싸는 태그, 2.홀로 쓰이는 태그로 구분된다.
어떤 태그를 잘 모르겠다면 구글에 'html OO tag'로 검색해보자.

통계에 기반한 학습

통계를 통해 빈번히 사용되는 태그들 위주로 학습하는 것이 효율적이다.
모르는 태그라 하더라도 1분이면 찾을 수 있기 때문이다.
그러니 여기에 나오는 통계를 활용하여 사용빈도가 높은 태그 위주로 학습해두자.

<br> vs <p>

html에게는 '기능 vs 의미'에서 승자는 의미이다.
웹에서 '접근성'은 매우 중요한 요소이다.
먼나라 이웃도, 시각장애인도 접근할 수 있어야 한다.
그러기에 문단 줄바꿈을 할 때 <br>이 아니라 <p>을 사용하는 것이 바람직하다.
이외에도 웹페이지를 작성할 때 항상 의미에 맞는 태그를 사용하도록 하자.

웹표준의 중요성

웹표준을 준수하며 알맞은 태그로 페이지를 작성해야 한다.
그래야 검색엔진에서 상위노출될 가능성이 높아지기 때문이다.
상위에 노출되기 위해 광고비를 지출할 필요가 없다.

최후의 문법, '속성' - <img>

이미지 태그 <img>에서 이미지 출처를 가져오려면 '속성'이라는 개념을 알아야 한다.
속성은 src=""의 형태로 작성한다.
속성 값을 큰따옴표로 입력한다는 점, 순서는 상관이 없다는 점이 특징이다.

tip으로 <img>의 중요한 속성은 alt="(이미지 설명)"이다.
시각장애인들도 사진에 대한 정보를 얻을 수 있기 때문에 검색엔진에게 좋은 점수를 받는다.

부모자식과 목록태그

속을 감싸고 있는 태그를 부모태그, 속을 자식태그라 부른다.
대표적인 태그는 목록태그, <ol>과 <ul>.

예시
<ul>
          <li> </li>
          <li> </li>
          <li> </li>
 </ul>

태그의 제왕 <a>

<a href="#">는 다른 웹페이지와 연결해주는 가장 중요한 태그이다.
새탭으로 열고 싶다면 target="_blank" 라는 속성을 추가하면 된다.
마우스를 올리기만 했을 때 정보를 표시하려면 title="(설명)" 속성을 추가하면 된다.

웹문서를 엮어 웹사이트를 만드는 법

앵커태그를 이용해서 사이트를 엮는다.
기본적으로 메인 페이지를 index.html로 하여 하위 페이지들을 연결시킨다.

서버와 클라이언트

서버(sever)는 말 뜻대로 '제공자'이다. 웹페이지를 요청자(client)에게 보내주는 서비스를 해주는 것이 서버이다.
그러므로 우리가 작성한 페이지를 웹 서버에 업로드 해놓아야 타인이 내 웹 페이지를 열람할 수 있다.
그렇게 하는데는 1.웹호스팅 방식 2. 웹서버를 직접 열기의 방법이 있다.하지만 보통 웹호스팅을 많이 이용한다.

웹 서버와 http

대부분의 웹페이지 주소 앞에 'https://'가 있는 것을 볼 수있다.
http란 hyper text(웹페이지) transfer(전송) protocol(약속)의 약자이다.
즉, 웹서버와 웹브라우저가 웹페이지를 서로 주고 받기 위한 약속을 의미한다.

웹 서버와 브러우저의 통신

수많은 웹페이지 중, 내가 원하는 웹페이지만을 받아내려면 주소가 필요하다.
그 주소가 바로 IP주소이다.
웹서버를 설치한 컴퓨터(본인 컴퓨터)의 ip주소를 확인하려면
'네트워크 오른쪽 클릭'-'네트워크 및 인터넷 설정'-'연결 속성 변경'에서 IPv4주소를 확인하면 된다.
